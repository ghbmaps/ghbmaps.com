Frequently Asked Questions

Select the building from the Building menu, then choose the Floor. The map zooms to the level and loads the floor layer automatically. If bookmarks are enabled, you can also pick a saved location for one-click access.

Use the global search bar. Start typing a name, jack number, or room ID—suggestions appear as you type. Select a result to zoom, pan, and open the details panel. If you’re on a floor app, search automatically scopes to the active level.

Open the Filters panel, toggle the category you need (e.g., Department), and apply one or more values. The map and results list update instantly. Click “Clear” to reset, or “Save” (if available) to keep your current filter set for later.

Click a feature or a search result to open its details panel. You’ll see profile info, assignment, related links (e.g., SharePoint, reassignment form), and any custom actions. Use the back arrow to return to your previous results.

Use the Home button to return to the default extent. Clear any active filters and close the results panel. If the app supports it, click the “Reset” UI action to restore the original map state in one step.

Open the Basemap widget to change context (e.g., imagery vs. streets). If the app includes a Scene view, use the 2D/3D toggle to switch. Your filters and selection carry over so you can keep working in the new view.

Use the Share widget to copy a link to your current extent, filters, and selection. If Bookmarks are enabled, add a bookmark for quick return later. Shared links open the app with the same state for your colleagues.

Yes—if URL parameters are configured. Append keys like ?bldg=JFK&floor=3&jack=J-1203 or ?q=Jane+Doe to deep-link into the app. Share the resulting URL to take others straight to the same view.

On small screens, panels collapse to save space—use the panel toggle to reveal filters or results. Pinch-zoom the map, and swipe the results list. For precise selection, zoom in and tap features to open the details card.

Open the Help or Feedback link in the header. Include the building, floor, and jack/room number, plus a screenshot if possible. Your request is routed to the support team and tracked so you can follow progress.
